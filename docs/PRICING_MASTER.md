| Rule | Description |
|-----|------------|

| Web shows monthly | Stripe charges annual real price |
| No API access | Only dataset access |

| Starter = 33% of Pro | Short-term is more expensive |
| Starter has NO updates | Pro has updates |

## PLAN DEFINITIONS

### OPEN DATA (FREE)

• Public dataset  
• Limited exports  
• No mass download  
• No premium fields  
• Unlimited use under open license  

---

### STARTER 90D (Full dataset, NO updates)

• 1 country  
• Full dataset dump at purchase  
• **No updates included**  
• Commercial internal use  
• No resale / sublicense  
• Valid for 90 days  

Price = Pro / 3

---

### PRO COUNTRY (Core product)

• 1 country  
• Full dataset  
• **Includes updates for 12 months**  
• Commercial internal use  
• No resale  

Annual recurring license

---

### AGENCY (Multi-country)

• Up to 5 countries  
• Full dataset  
• Includes updates  
• Use for client services  
• Multi-seat allowed  
• Annual license  

---

### ENTERPRISE (Corporate contract)

• Unlimited countries  
• SLA & compliance  
• Annual contract
